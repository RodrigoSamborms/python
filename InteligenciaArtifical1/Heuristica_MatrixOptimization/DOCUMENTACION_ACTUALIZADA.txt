📋 REFERENCIA RÁPIDA INLINE
════════════════════════════════════════════════════════════════════════════

OPERADORES:
  @  → Multiplicación matricial (ej: A @ B)
  +  → Suma (ej: A + B)
  -  → Resta (ej: A - B)

REGLAS:
  • Variables: letras, números, guiones bajos (ej: A, M1, x_2)
  • Precedencia: @ > +/- (use paréntesis para cambiar)
  • Espacios: opcionales (A@B = A @ B)

EJEMPLOS COMUNES:
  (A @ B) + (C @ D)     → Dos multiplicaciones independientes
  A @ B @ C + D         → Cadena con suma
  (A @ B + C) @ D       → Suma luego multiplicación
  ((A + B) @ C)         → Suma primero, luego multiplicación

EXPRESIONES A EVITAR:
  A B                   → Falta operador (usar @ o +)
  (A @ B                → Paréntesis no cerrado
  A @ @ B               → Operador doble
  1A @ B                → Variable no puede comenzar con número
